# Movie-Records
A movie records dataset containing information such as titles, genres, release years, ratings, and revenue, used to analyze trends, performance, and audience preferences in the film industry

**Key Objectives**
- Identify top-rated movies
- Analyze average ratings by genre
- Examine movie release trends over time

**Data Set**
- Title
- Genre
- Release Year
- Rating
- Revenue

**Tools**
  - SQL
  - Excel
  - Python

**Key Insight**
Movie ratings vary by genre, and higher revenue does not always mean higher ratings.
